Question 3:

Method 1 uses callbacks while method 2 uses async/await which is based on promises.

The console output of both functions will be the same, and neither of them explicitly return anything. But, since method 2 is an async function, it implicity returns a promise. Therefore we can add a .then() statement afterwards if we want to queue up some code to be run after method 2 has completed.

Since method 1 is a regular function we have no way to tell whether it has completed or not.
